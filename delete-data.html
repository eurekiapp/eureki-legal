Pedido de eliminação de dados (sem apagar a conta)

Aqui encontras as opções para eliminar dados na app (ex.: notas, histórico de chat), mantendo a tua conta ativa.

O que podes eliminar

  • Notas criadas na app.
  • Histórico de chat (conversas com a mentora).
  • Dados de progresso/objetivo (ex.: objetivo atual, preferências, micro-passos guardados).
  • Conteúdo criado pelo utilizador dentro da app (quando aplicável).

Como eliminar dentro da app (recomendado)

1

Eliminar histórico do chat

Abre o chat → menu (⋯) → Apagar histórico.

2

Eliminar notas

Vai a Notas → seleciona uma nota → Apagar (ou “Apagar tudo” se existir essa opção).

3

Eliminar dados do objetivo/progresso

Vai a Definições → Privacidade / Dados → Apagar dados do objetivo (o nome da opção pode variar).

Eliminar dados na app é imediato e não requer contacto com suporte.

Se não conseguires eliminar na app

Podes pedir a eliminação de dados por email. Para acelerar o processo, envia:

  • Assunto: Pedido de eliminação de dados — Eureki
  • O email da conta (ou o método de login usado: Google, email/password).
  • O que queres apagar: chat, notas, progresso, ou tudo exceto conta.
  • Opcional: uma breve descrição (ex.: “apagar todas as notas e histórico de chat”).
Enviar email para suporte Quero apagar a conta (e tudo)

Importante: por motivos de segurança, poderemos pedir confirmação adicional para garantir que és o titular da conta.

Prazos e retenção

Quando eliminas dados na app, a remoção é imediata no dispositivo e, quando aplicável, é propagada para os sistemas associados.

Em pedidos feitos por email, o tratamento pode requerer validação do titular.

Alguns registos técnicos (ex.: falhas/crashes) podem ser mantidos por um período limitado para segurança/diagnóstico.
